This code runs a Go/No-Go experiment using PsychoPy. The program loads a list of food images from a CSV file and randomly selects stimuli for the task. During the experiment, participants complete 50 trials in which a fixation cross appears briefly, followed by a food image and a cue. Participants are instructed to press the space bar on Go trials and withhold their response on No-Go trials. The program records the participant’s response, reaction time, and accuracy for each trial, and saves the results to a CSV file at the end of the experiment.
